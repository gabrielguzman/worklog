## Task Management

### Creating Tasks
1. Click **"Nueva tarea"** or press `Cmd+N`
2. Fill in:
   - **Title** (required)
   - **Description** (optional)
   - **Priority**: Urgent, High, Medium, Low
   - **Due Date**
   - **Project** (optional)
   - **Tags** (optional)
3. Click **"Crear tarea"**

### Task Operations
- **Edit**: Click task → Update → Save
- **Complete**: Check checkbox or click status dropdown
- **Recur**: Set daily/weekly/monthly with end date
- **Delete**: Click task → Delete button
- **Reorder**: Drag to reorder (Kanban view)

### Kanban View
- See tasks organized by status (Pending → In Progress → Done)
- Drag tasks between columns to update status
- Filter by project, priority, or tag
- Quick actions on hover

### Task Statuses
- 🟡 **Pending**: Not started
- 🔵 **In Progress**: Currently working on it
- 🟢 **Done**: Completed

### Task Priorities
- 🔴 **Urgent**: Do immediately
- 🟠 **High**: Important
- 🟡 **Medium**: Normal
- ⚪ **Low**: Nice to have

## Time Tracking

### Logging Entries
1. Click **"Nueva entrada"** or press `Cmd+Shift+N`
2. Choose entry type:
   - 📝 **General**: Regular work
   - 🤝 **Meeting**: Meeting/discussion
   - 🚀 **Deploy**: Deployment/release
   - 🔍 **Code Review**: Reviewing code
   - 🔬 **Research**: Investigation/learning
   - 📋 **Planning**: Planning/design

3. Fill in:
   - **Title** (required)
   - **Content**: Details/notes (supports Markdown)
   - **Date & Time**
   - **Project** (optional)
   - **Tags** (optional)

4. Click **"Crear entrada"**

### Entry Types
Each type helps categorize your work. Use consistently for better analytics.

### Markdown Support
Format your entry content:
- **Bold**: `**text**`
- *Italic*: `_text_`
- `Code`: `` `code` ``
- Lists: `- item`
- Headers: `## Header`

### Focus Sessions (Pomodoro)
1. Click a task
2. Click **"Iniciar sesión de enfoque"**
3. Session runs in background
4. Complete when done

## Tips & Best Practices

### Productivity Tips
1. **Use Projects** to organize work by context
2. **Add Tags** for cross-project filtering
3. **Set realistic due dates** to reduce stress
4. **Use focus sessions** for deep work (pomodoros)
5. **Review weekly reports** to identify patterns
6. **Check daily AI summary** for context

### Time Tracking Tips
1. **Log entries daily** - don't wait til end of week
2. **Be specific** with entry titles
3. **Use entry types** to categorize work
4. **Add notes** about decisions or blockers
5. **Link to projects** for better analytics

### Task Management Tips
1. **Prioritize ruthlessly** - don't mark everything urgent
2. **Break down** big tasks into subtasks
3. **Use recurrence** for repeating work
4. **Set due dates** but be realistic
5. **Complete tasks** to build momentum

### Dashboard Tips
1. **Check metrics daily** - see your productivity
2. **Review weekly** - spot trends and patterns
3. **Use AI summary** - get daily context
4. **Monitor focus time** - ensure breaks
5. **Adjust** based on what you learn

### Mobile Tips
1. Use **quick capture** for fast entry
2. Access **kanban** for quick updates
3. Check **dashboard** for daily summary
4. Use **dark mode** in low light

## Troubleshooting

### Task not saving?
- Check internet connection
- Verify all required fields filled
- Try refreshing page
- Check browser console for errors

### Entries not appearing?
- Refresh page (Cmd+R)
- Check correct date selected
- Verify you're in correct project

### Charts not loading?
- Refresh page
- Try different week/month
- Disable ad blockers
- Try different browser

### Focus session not working?
- Ensure browser tab stays open
- Check notifications enabled
- Try on desktop instead of mobile

### Dark mode issues?
- Clear browser cache
- Try toggling on/off
- Check browser dark mode setting
- Refresh page
